아고다 보장제 활용 방법

아고다를 통해 호텔 객실을 예약하신 후 타 사이트에서 동일한 객실을 동일한 숙박 날짜 및 조건에 더 낮은 요금으로 예약 가능함을 발견하신 경우 당사의 단독 재량으로 해당 요금에 맞춰드리거나, 차액을 아고다 계정의 AgodaCash로 적립해 드리고 있습니다.

아고다 캐시백리워드로 페이백 받기

아고다에서 예약한 호텔 중 리워드가 가능한 숙소라면 퇴실 60일 이후 캐시백 리워드 신청이 가능합니다. 예약 시 작성한 이메일을 확인하여 신청하면 됩니다.

아고다 후지불 결제 시 수수료가 발생하나요?

아고다 나중에 결제(후지불)를 선택하는 경우, 결제일 당시 블룸버그 환율에 5%를 추가하여 최종 결제 금액이 올라갈 확률이 높습니다.

세금 및 봉사료가 포함된 금액으로 호텔을 확인하고 싶어요.

네. 사이트 우측 상단 ‘요금 보기 설정’에서 1박당 요금(세금 및 봉사료 포함)으로 설정 하시면 됩니다.

솔라 쿠폰을 이용하는 경우 불이익이 있나요?

솔라 쿠폰을 이용하여 아고다를 할인받는 경우 얻는 불이익은 전혀 없습니다.

아고다 할인코드 사용 방법

1

원하는 숙소 검색 후 선택

아고다(agoda.com) 또는 앱에서 여행지·날짜를 입력하고, 원하는 호텔·숙소를 검색한 뒤 상품을 선택합니다.

2

결제 단계에서 프로모션 코드 입력

예약 정보를 입력한 뒤 결제 페이지로 이동합니다. ‘프로모션 코드’ 또는 ‘할인코드’ 입력란에 코드를 넣고 적용 버튼을 누르면 할인이 반영됩니다.

할인코드 사용이 안되는 경우

1

유효기간·적용 대상 확인

쿠폰 유효기간이 지나지 않았는지, 해당 프로모션이 선택한 숙소·상품에 적용되는지 확인하세요. 일부 코드는 국내/해외·특정 호텔만 대상입니다.

2

최소 금액·중복 적용 여부 확인

최소 예약 금액 조건을 충족하는지 확인하고, 이미 다른 프로모션 코드를 사용 중이면 해제 후 다시 입력해 보세요. 아고다는 예약 1건당 하나의 프로모션 코드만 적용됩니다.
