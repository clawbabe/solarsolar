다이슨 혜택

쿠폰코드 — 공식몰 프로모션 페이지에서 NEWYEARBOK, NEWYEARBOK2 등 코드 확인 후 장바구니에서 입력하면 10%·10만원 할인 적용. 제품별 적용 대상·기간이 다르므로 상세페이지를 꼭 확인하세요.

카드·간편결제 — 하나·삼성·롯데 카드 즉시 5~10% 할인, 롯데 12만원 캐시백, 삼성·현대 포인트, 토스 5%·카카오페이 7% 적립 등. 최소 결제 금액·예산 소진 시 조기 종료에 유의.

무이자·리뷰 — 25~30만원 이상 결제 시 카드사별 12~36개월 무이자 할부. 네이버페이 결제 후 포토 리뷰 작성 시 3만~5만 포인트·액세서리 바우처 지급 이벤트도 자주 진행됩니다.

자주 묻는 질문

다이슨 쿠폰은 어떻게 사용하나요?

다이슨 공식몰(www.dyson.co.kr)에서 상품을 장바구니에 담은 뒤, 결제 단계의 쿠폰 입력란에 NEWYEARBOK·NEWYEARBOK2 등 코드를 입력하면 할인이 적용됩니다. 카드·간편결제 할인은 해당 결제 수단 선택 시 자동 적용되며, 무이자 할부·리뷰 이벤트는 조건 충족 시 결제 화면에서 선택하거나 구매 후 리뷰 작성으로 혜택을 받습니다.

쿠폰코드와 카드 할인을 같이 쓸 수 있나요?

일반적으로 다이슨 공식몰에서는 쿠폰코드 할인과 다른 쿠폰의 중복 사용이 불가하다고 안내합니다. 카드 즉시할인·캐시백·포인트 적립은 프로모션별로 중복 가능 여부가 다르며, 무이자 할부는 할인 적용 후 결제금액 기준으로 이용 가능합니다. 결제 화면에서 적용 가능한 혜택이 자동 표시되므로 확인 후 결제하세요.

무이자 할부는 어떤 카드로 받나요?

롯데카드는 25만 원 이상~300만 원 미만 결제 시 최대 12개월, 현대·하나·삼성카드는 30만 원 이상~300만 원 미만 결제 시 최대 12개월 무이자 할부가 가능합니다. 삼성카드 로봇청소기 전용 36개월 무이자도 있습니다. 법인·가족·선불·간편결제는 제외되며, 기간·한도는 프로모션 안내를 확인하세요.

할인코드가 적용되지 않아요.

쿠폰 유효기간(2월 9일~22일 등), 적용 대상 제품(에어랩 코안다2x, 슈퍼소닉 뉴럴 등), 1주문 1회·다른 쿠폰과 중복 불가 조건을 확인해주세요. 카드·간편결제 할인은 해당 수단으로 결제할 때만 적용되며, 예산 소진 시 조기 종료될 수 있습니다. 리뷰 이벤트는 네이버페이 결제·포토 리뷰 작성이 필수인 경우가 많습니다.

리뷰 이벤트 포인트·바우처는 언제 지급되나요?

새해맞이 리뷰 이벤트의 네이버페이 포인트는 2026년 3월 31일 지급 등으로 안내되는 경우가 있습니다. 구매 기간·리뷰 작성 기간을 지키고, 포토 리뷰(직접 촬영)·네이버페이 결제 조건을 충족해야 하며, 반품 시 참여가 무효될 수 있습니다. 액세서리 바우처는 리뷰 참여 완료 후 지급되며, 공식몰 액세서리 구매 시 사용 가능합니다.

다이슨 할인코드 사용 방법

1

다이슨 공식몰(www.dyson.co.kr)에서 상품 장바구니에 담기

할인 적용 대상 제품(에어랩, 슈퍼소닉, 청소기 등)을 장바구니에 넣습니다.

2

결제 단계에서 쿠폰 입력란에 코드 입력

NEWYEARBOK, NEWYEARBOK2 등 프로모션 코드를 입력하면 10%·10만 원 할인이 적용됩니다. 카드·간편결제 할인은 해당 결제 수단 선택 시 자동 적용됩니다.

3

무이자 할부·리뷰 이벤트 조건 확인

무이자 할부는 결제 금액·카드사에 따라 12~36개월 중 선택합니다. 리뷰 이벤트는 네이버페이 결제 후 포토 리뷰 작성으로 포인트·바우처를 받습니다.

4

할인·무이자 적용 금액 확인 후 결제

최종 결제 금액에 쿠폰·카드 할인·무이자가 반영되었는지 확인한 뒤 결제를 완료합니다.

할인코드 사용이 안 되는 경우

1

쿠폰 유효기간·적용 대상 제품 확인

쿠폰별 유효기간(2월 9일~22일 등)과 적용 제품(에어랩 코안다2x, 슈퍼소닉 뉴럴 등)이 맞는지 확인하세요. 다른 쿠폰과 중복 불가인 경우가 많습니다.

2

카드·간편결제 할인: 결제 수단·최소 금액 확인

해당 카드·토스·카카오페이로 결제했는지, 최소 30만 원 등 조건을 충족했는지 확인합니다. 예산 소진 시 조기 종료될 수 있습니다.

3

무이자: 법인·가족·간편결제 제외 확인

무이자 할부는 개인 신용카드(일부 체크 가능)로만 적용됩니다. 법인·가족·선불·간편결제는 제외되므로 해당 카드로 결제했는지 확인하세요.

4

리뷰 이벤트: 네이버페이 결제·포토 리뷰 필수 확인

리뷰 포인트·바우처는 네이버페이 결제 후 포토 리뷰 작성이 필수입니다. 기간·대상 제품을 지켰는지 확인하세요.
