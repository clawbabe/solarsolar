마이리얼트립에서 제공되는 특가 상품 및 혜택인 경우 결제 단계 전, 쿠폰 사용 가능 여부에 대해 확인 해주세요.

솔라 할인코드는 모든 결제 수단에서 적용 가능한가요?

일반 계좌이제 및 간편결제, 카드결제 등 다양한 결제 수단에서도 사용 가능하며 해외에서도 결제 시 할인을 적용받으실 수 있습니다. 만약 적용이 안되는 경우, 고객지원실로 문의해주세요.

마이리얼트립 할인코드 사용 방법

1

원하는 투어/액티비티 선택하기

마이리얼트립 웹사이트 또는 앱에서 원하는 여행 상품을 검색하고 선택합니다.

2

예약 페이지에서 프로모션 코드 입력

결제 정보 입력 페이지에서 프로모션 코드 또는 할인코드 입력란을 찾아 코드를 입력하고 적용합니다.

3

할인 적용 확인 후 결제

할인코드 사용이 안되는 경우

1

유효기간 확인

쿠폰의 유효기간이 지나지 않았는지 확인하세요. 만료된 코드는 사용할 수 없습니다.

2

최소 주문금액 확인

일부 할인코드는 최소 주문금액 조건이 있습니다. 예약 금액이 조건을 충족하는지 확인하세요.

3

적용 가능한 상품 확인

일부 코드는 특정 투어/액티비티에서만 사용 가능합니다. 쿠폰 사용 조건을 확인하세요.

4

중복 적용 여부 확인

마이리얼트립에서는 한 번의 예약에 하나의 프로모션 코드만 적용 가능합니다. 이미 다른 코드를 적용했는지 확인하세요.
