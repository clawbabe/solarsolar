할인쿠폰의 유효기간은 어떻게 되나요?

할인쿠폰의 유효기간은 1주일부터 4-6주까지 다양하므로, 사용 전 만료일을 반드시 확인해야 합니다.

다른 할인과 중복 사용할 수 있나요?

대부분의 호텔스닷컴 할인쿠폰은 다른 프로모션이나 회원 할인과 중복 사용이 불가능합니다.

쿠폰 사용 시 호텔 포인트 적립이 가능한가요?

네, 할인쿠폰 사용 시에도 호텔 회원 포인트 적립이 가능합니다.

할인쿠폰 사용 시 주의사항은 무엇인가요?

쿠폰 코드 입력, 유효기간 및 중복 사용 가능 여부를 사전에 확인해야 합니다.

2

예약 페이지에서 프로모션 코드 입력

3

할인코드 사용이 안되는 경우

1

유효기간 확인

쿠폰의 유효기간이 지나지 않았는지 확인하세요. 만료된 코드는 사용할 수 없습니다.

2

최소 주문금액 확인

일부 할인코드는 최소 주문금액 조건이 있습니다. 예약 금액이 조건을 충족하는지 확인하세요.

3

적용 가능한 호텔 확인

일부 코드는 특정 호텔이나 지역에서만 사용 가능합니다. 쿠폰 사용 조건을 확인하세요.

4

중복 적용 여부 확인
