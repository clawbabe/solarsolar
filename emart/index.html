자주 묻는 질문

이마트 인터넷 쇼핑몰 할인코드는 어떻게 사용하나요?

이마트 인터넷 쇼핑몰 앱 또는 웹에서 상품을 담고 결제 단계로 이동하세요. 쿠폰함에서 다운로드한 쿠폰을 적용하거나, 결제 시 쿠폰 코드 입력란에 코드를 입력하면 할인이 적용됩니다.

여러 쿠폰을 함께 사용할 수 있나요?

상품당 1개 쿠폰 적용 가능하며, 장바구니 쿠폰은 중복 사용이 불가할 수 있습니다. 각 쿠폰의 유의사항을 확인하세요.

할인코드가 적용되지 않아요.

유효기간, 최소 구매 금액, 회원 전용 여부, ID당 사용 횟수를 확인해주세요.

이마트 인터넷 쇼핑몰 할인코드 사용 방법

1

쿠폰 다운로드

이마트 인터넷 쇼핑몰 앱 또는 웹에서 쿠폰함·프로모션 페이지에서 원하는 쿠폰을 다운로드합니다.

2

상품 담기 후 결제

해당 브랜드/카테고리 상품을 장바구니에 담고 결제 단계로 이동합니다.

3

쿠폰 적용 후 결제

결제 페이지에서 적용 가능한 쿠폰을 선택하고 할인 적용 후 결제합니다.

할인코드 사용이 안 되는 경우

1

유효기간·최소 금액 확인

쿠폰 유효기간과 최소 구매 금액을 확인하세요. 만료되었거나 조건 미충족 시 적용되지 않습니다.

2

회원 전용 여부

일부 쿠폰은 이마트 인터넷 쇼핑몰 회원 전용입니다. 로그인 또는 회원가입 후 이용하세요.

3

적용 대상 상품 확인

브랜드·카테고리 제한이 있는 쿠폰은 해당 상품에서만 사용 가능합니다.
