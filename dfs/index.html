쿠폰의 유효기간은?

2025년 6월 30일까지 사용 가능하며, 유효기간 만료 시 솔라쿠폰에서 최신 업데이트 합니다.

중복 사용이 가능한가요?

여권 당 1일 1회 1매 사용 가능합니다.

전자담배 구매 시 할인 적용 가능한가요?

전자담배를 포함한 일부 품목은 사용 불가능하며 자세한 내용은 이미지 내 안내사항을 참고해주세요.

일행과 함께 사용 가능한가요?

네. 쿠폰은 일행과 사용 가능하지만 여권 당 1일 1회 사용 가능합니다.

면세점 할인코드 사용 방법

1

원하는 상품 선택하기

면세점 웹사이트 또는 앱에서 원하는 상품을 검색하고 장바구니에 담습니다.

2

결제 페이지에서 쿠폰 입력

결제 페이지에서 쿠폰 코드 입력란을 찾아 코드를 입력하고 적용합니다.

3

할인 적용 확인 후 결제

할인된 금액을 확인하고 출국 정보와 함께 결제를 완료합니다.

할인코드 사용이 안되는 경우

1

유효기간 확인

쿠폰의 유효기간이 지나지 않았는지 확인하세요. 만료된 코드는 사용할 수 없습니다.

2

최소 주문금액 확인

일부 할인코드는 최소 주문금액 조건이 있습니다. 주문 금액이 조건을 충족하는지 확인하세요.

3

적용 가능한 카테고리 확인

일부 코드는 특정 카테고리에서만 사용 가능합니다. 쿠폰 사용 조건을 확인하세요.

4

중복 적용 여부 확인

면세점에서는 한 번의 주문에 하나의 쿠폰만 적용 가능합니다. 이미 다른 쿠폰을 적용했는지 확인하세요.
